FireHOL 
FireHOL is a Linux iptables stateful packet filtering firewall generator. It's goal is to be extremely abstracted so that the administrators can design firewalls of any complexity without the need to know all the details about protocols, sockets, ports, etc. FireHOL has been designed to be a firewall... for humans...

FireHOL is distributed under GPL.
